# 源代码目录

本目录包含项目的所有源代码文件。

## 目录结构

- **hls-service/** - HLS-Communication核心服务代码
  - 独立的后台进程，负责设备通信
  - 使用C#/.NET Core开发
  - 提供IPC接口供Node-RED调用

- **nodes/** - Node-RED定制节点代码
  - hls-read节点：数据采集节点
  - hls-write节点：设备控制节点
  - 使用Node.js开发

## 开发规范

- 遵循各语言的最佳实践和编码规范
- 所有代码都需要通过代码质量检查
- 重要模块需要编写单元测试
- 提交前需要运行完整的测试套件
